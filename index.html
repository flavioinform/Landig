Git y GitHub

Sistema de Control de Versiones

Git es un sistema distribuido que rastrea cambios en archivos y coordina el trabajo entre múltiples personas.

Colaboración Global

GitHub es una plataforma que permite alojar repositorios Git y colaborar con desarrolladores de todo el mundo.

Código Abierto

Creado por Linus Torvalds en 2005, Git es completamente gratuito y de código abierto.

Comandos Esenciales

1 Configurar Usuario 

git config --global user.name "Tu Nombre"
git config --global user.email "tuemail@ejemplo.com"

Configura tu identidad para que Git sepa quién hace los cambios.

2 Inicializar Repositorio 

git init

Crea un nuevo repositorio Git en el directorio actual.

3 Agregar Archivos 

git add .

Prepara todos los archivos modificados para el siguiente commit.

4 Confirmar Cambios 

git commit -m "mensaje del commit"

Registra los cambios agregados con un mensaje descriptivo.

5 Subir al repositorio 

git push
